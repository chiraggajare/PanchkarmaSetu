# Assignment 3: Django Admin Configuration

## Description
This assignment covers the activation of the Django admin interface, the registration of models with the admin, and the customization of the interface.

## Files Involved

### 1. Admin Logic
*   **core/admin.py**: Definitions of `CustomUserAdmin` and registration of all other models.

### 2. Model Registration
*   **core/models.py**: Used as a data source for the admin view.

### 3. URL Mapping
*   **panchkarma_setu/urls.py**: Root path for the admin site (`admin/`).

## Key Code Snippets

### Custom Admin (core/admin.py)
```python
class CustomUserAdmin(UserAdmin):
    list_display = ('username', 'email', 'role', 'is_staff')
    fieldsets = UserAdmin.fieldsets + (
        (None, {'fields': ('role',)}),
    )

admin.site.register(User, CustomUserAdmin)
```

### Model Registration (core/admin.py)
```python
admin.site.register(TreatmentPlan)
admin.site.register(Appointment)
# ... other models
```
