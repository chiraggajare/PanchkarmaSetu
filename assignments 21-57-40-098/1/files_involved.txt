# Assignment 1: Project Setup, URLs, and Basic Templates

## Description
This assignment covers the initial setup of the Django project and the `core` application, the creation of a basic landing page, and URL mapping.

## Files Involved

### 1. Project Configuration
*   **panchkarma_setu/settings.py**: Main project settings (Installed Apps: `core`, Database: `sqlite3`).
*   **panchkarma_setu/urls.py**: Root URL configuration, including `core.urls`.

### 2. App Structure
*   **core/apps.py**: Core application definition.
*   **core/urls.py**: App-specific URL patterns.

### 3. Templates & Static Content
*   **templates/base.html**: Base layout template.
*   **templates/core/home.html**: Public landing page.
*   **static/css/styles.css**: Global styling.
*   **static/css/home.css**: Landing page specific styling.

### 4. Views & Dynamic Data
*   **core/views.py**: `home` view rendering the landing page with dynamic statistics.

## Key Code Snippets

### URL Mapping (core/urls.py)
```python
urlpatterns = [
    path('', views.home, name='home'),
    # ... other paths
]
```

### Context Rendering (core/views.py)
```python
def home(request):
    patients_treated = User.objects.filter(role='patient').count()
    return render(request, 'core/home.html', {'patients_treated': patients_treated})
```
